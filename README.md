# Usage
<!-- usage -->
```sh-session
$ npm install -g webaily-cli
$ webaily COMMAND
running command...
$ webaily (-v|--version|version)
webaily-cli/0.0.4 win32-x64 node-v10.15.1
$ webaily --help [COMMAND]
USAGE
  $ webaily COMMAND
...
```
